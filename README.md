# Create Test from Flashcards CSV
## Setup
In the folder you have this repository add a folder called "data" and the CSV's with the terms and definitions you want. The CSV's should be formated like:

```{csv}
term1,def1
term2,def2
term3,def3
```

## Running
Simply run the python file and it will ask which data set (from the data folder) you want to be tested on. Once you seleect a flashcard set, it will then ask how many of each multiple choice question you want and how many options you want to choose from.
